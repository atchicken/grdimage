# grdimage
Draw Using the GMT "grdimage" Command

## How to Use

Create NetCDF File from [netcdf](https://github.com/atchicken/netcdf) Repository

This is Assuming Global Temperature Data

Then Run the Code Below

```
./grdimage.sh
```
